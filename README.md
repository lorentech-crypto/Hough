Cálculo de histogramas y detección de líneas con el algoritmo de Hough

Para la elaboración del programa se proporciona la imagen «secreto.jpg» que contiene información oculta a simple vista. Es una imagen en escala de grises.

Implementar un programa en Python con la librería OpenCV que realice las siguientes operaciones:

▸	Leer la imagen «secreto.jpg» y mostrarla en una ventana.

▸	Ecualizar la imagen para que se vea claramente su contenido y mostrar el resultado en una ventana.

▸	Aplicar el algoritmo de detección de líneas de Hough sobre la imagen ecualizada para detectar las líneas más largas. Filtrar las líneas detectadas para eliminar las «similares» (líneas muy próximas en rho y theta).   

▸	Dibujar   las líneas resultantes en color blanco sobre una copia de la imagen ecualizada y mostrar el resultado en una ventana.

▸	Mostrar por consola el mensaje: «Se han dibujado x líneas», donde x es el número total de líneas realmente dibujadas.

▸	Cualquier imagen intermedia en el proceso de resolución de la actividad se puede mostrar en una ventana si se considera relevante (por ejemplo, la imagen con bordes detectados antes de la aplicación del algoritmo de Hough).

▸	Incluir comentarios claros y descriptivos en el código para explicar los pasos seguidos y las decisiones tomadas en el diseño de la solución.
